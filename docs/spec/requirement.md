# 要件定義書: 2次元イジングモデル モンテカルロシミュレーション (Ising2D)

## 1. プロジェクト概要

### 1.1 目的
2次元正方格子上のイジングモデルに対して、メトロポリス法によるモンテカルロシミュレーションを実施し、相転移現象の解析・可視化を行うJuliaベースのシミュレーションツールを開発する。

### 1.2 研究背景
- イジングモデルは統計力学における最も基本的な格子模型であり、強磁性体の相転移を記述する
- 2次元イジングモデルはOnsager (1944) により厳密解が知られており、数値シミュレーションの検証に適している
- メトロポリス法はマルコフ連鎖モンテカルロ (MCMC) の代表的なアルゴリズムであり、詳細釣り合い条件を満たす
- 有限サイズスケーリング解析により、有限系のシミュレーション結果から熱力学的極限の臨界指数を推定できる

### 1.3 対象システム
- **モデル**: 2次元正方格子上のイジングモデル
- **格子サイズ**: L x L（L = 8, 16, 32, 64, 128 など）
- **スピン変数**: s_i = +1 または -1
- **境界条件**: 周期的境界条件 (Periodic Boundary Conditions)
- **ハミルトニアン**:
  ```
  H = -J Σ_{<i,j>} s_i s_j - h Σ_i s_i
  ```
  ここで、J は交換相互作用定数、h は外部磁場、Σ_{<i,j>} は最近接対の和

## 2. 機能要件

### 2.1 格子・スピン系の構成 (F-LAT)

#### F-LAT-01: 正方格子の生成
**概要**: L x L の2次元正方格子を生成し、周期的境界条件を適用する

**仕様**:
- 格子サイズ L を指定して正方格子を生成
- 各格子点にスピン変数 s_i ∈ {+1, -1} を配置
- 周期的境界条件により、格子の端が反対側の端と接続される

**受け入れ条件**:
- [ ] 任意の正整数 L に対して正方格子を生成可能
- [ ] 周期的境界条件が正しく実装されている
- [ ] 各スピンが正確に4つの最近接格子点を持つ

#### F-LAT-02: 初期スピン配置の設定
**概要**: シミュレーション開始時のスピン配置を設定する

**初期配置オプション**:
| オプション | 説明 | 用途 |
|-----------|------|------|
| 全スピン上向き (all-up) | s_i = +1 (∀i) | 低温からの緩和過程の観察 |
| 全スピン下向き (all-down) | s_i = -1 (∀i) | 対称性の確認 |
| ランダム配置 (random) | s_i = ±1 を等確率で設定 | 高温からの冷却過程の観察 |

**受け入れ条件**:
- [ ] 3種類の初期配置が選択可能
- [ ] ランダム配置で乱数シードの設定が可能

---

### 2.2 モンテカルロシミュレーション (F-MC)

#### F-MC-01: メトロポリス法の実装
**概要**: メトロポリス・ヘイスティングスアルゴリズムにより、カノニカル分布に従うスピン配置のサンプリングを行う

**アルゴリズム**:
1. 格子上のスピン i をランダムに選択
2. スピン反転時のエネルギー変化 ΔE を計算:
   ```
   ΔE = 2 s_i (J Σ_{j ∈ nn(i)} s_j + h)
   ```
   ここで nn(i) はスピン i の最近接格子点の集合
3. 遷移確率 W を計算:
   ```
   W = min(1, exp(-ΔE / (k_B T)))
   ```
4. 一様乱数 r ∈ [0, 1) を生成し、r < W ならスピンを反転: s_i → -s_i

**1モンテカルロステップ (MCS)** = L x L 回のスピン反転試行

**パラメータ**:
| パラメータ | 記号 | デフォルト値 | 説明 |
|-----------|------|-------------|------|
| 交換相互作用 | J | 1.0 | 正: 強磁性、負: 反強磁性 |
| 外部磁場 | h | 0.0 | 外部磁場の強さ |
| 温度 | T | 可変 | シミュレーション温度 (k_B T/J の単位) |
| 格子サイズ | L | 32 | 正方格子の辺の長さ |

**受け入れ条件**:
- [ ] メトロポリス法が詳細釣り合い条件を満たしている
- [ ] ΔE の計算が周期的境界条件を考慮している
- [ ] 高温極限で磁化が0に近づく
- [ ] 低温極限で磁化が±1に近づく

#### F-MC-02: エネルギー変化の高速計算
**概要**: ΔE は h = 0 の場合 {-8J, -4J, 0, +4J, +8J} の5通りしかとらないため、exp(-ΔE/(k_B T)) を事前計算してルックアップテーブルとして利用する

**受け入れ条件**:
- [ ] ルックアップテーブルが正しく構築される
- [ ] ルックアップテーブル使用時と直接計算時で同一の結果が得られる

#### F-MC-03: 熱平衡化
**概要**: 初期配置から熱平衡状態に到達するまでの緩和過程を管理する

**仕様**:
- 熱平衡化ステップ数 n_eq を設定可能（デフォルト: 1000 MCS）
- 熱平衡化中はサンプリングを行わない

**受け入れ条件**:
- [ ] 熱平衡化ステップ数を指定可能
- [ ] 熱平衡化後のデータのみが物理量計算に使用される

#### F-MC-04: 温度掃引シミュレーション
**概要**: 指定された温度範囲をスキャンし、各温度での物理量を計算する

**仕様**:
- 温度範囲 [T_min, T_max] と温度刻み ΔT を指定
- 臨界温度 T_c = 2J/(k_B ln(1+√2)) ≈ 2.269J/k_B 付近を十分な解像度でカバー
- 各温度で独立にシミュレーションを実行

**受け入れ条件**:
- [ ] 温度範囲と刻みを指定可能
- [ ] 各温度で独立にシミュレーションが実行される

---

### 2.3 物理量の計測 (F-OBS)

#### F-OBS-01: エネルギーの計算
**数学的定義**:
```
E = -J Σ_{<i,j>} s_i s_j - h Σ_i s_i
e = E / N    (N = L^2)
```

**受け入れ条件**:
- [ ] 全スピン上向き配置で E = -2JN（h = 0 の場合）
- [ ] 周期的境界条件でボンドの重複カウントがない

#### F-OBS-02: 磁化の計算
**数学的定義**:
```
M = Σ_i s_i
m = M / N
```

**出力**: ⟨|m|⟩, ⟨m²⟩, ⟨m⁴⟩（ビンダーキュムラント計算用）

**受け入れ条件**:
- [ ] 全スピン上向き配置で m = 1
- [ ] 高温極限で ⟨|m|⟩ → 0
- [ ] T < T_c で ⟨|m|⟩ > 0（自発磁化）

#### F-OBS-03: 比熱の計算
**数学的定義**:
```
C = (N/(k_B T²)) (⟨e²⟩ - ⟨e⟩²)
```

**受け入れ条件**:
- [ ] 臨界温度付近で比熱のピークが観測される
- [ ] 格子サイズ増大に伴いピーク値が増大する（対数的）

#### F-OBS-04: 磁化率の計算
**数学的定義**:
```
χ = (N/(k_B T)) (⟨m²⟩ - ⟨|m|⟩²)
```

**受け入れ条件**:
- [ ] 臨界温度付近で磁化率のピークが観測される

#### F-OBS-05: ビンダーキュムラントの計算
**数学的定義**:
```
U_L = 1 - ⟨m⁴⟩ / (3⟨m²⟩²)
```

**性質**: T → 0 で U_L → 2/3、T → ∞ で U_L → 0、T = T_c で異なる L の U_L が交差

#### F-OBS-06: 統計平均とエラー推定
- サンプリング間隔の設定（自己相関低減）
- ブロック平均法（ビニング法）による統計誤差の推定

---

### 2.4 有限サイズスケーリング解析 (F-FSS)

#### F-FSS-01: 臨界温度の推定
ビンダーキュムラントの交差点から臨界温度を推定。厳密解: T_c = 2J / (k_B ln(1 + √2)) ≈ 2.269185 J/k_B

#### F-FSS-02: 臨界指数の推定
**2次元イジングモデルの厳密な臨界指数**:
| 臨界指数 | 記号 | 厳密値 |
|---------|------|--------|
| 秩序変数の指数 | β | 1/8 |
| 磁化率の指数 | γ | 7/4 |
| 比熱の指数 | α | 0 (対数発散) |
| 相関長の指数 | ν | 1 |

---

### 2.5 可視化 (F-VIS)

#### F-VIS-01: スピン配置の可視化
- L x L 格子上のスピン配置のヒートマップ表示

#### F-VIS-02: 物理量の温度依存性
- e(T), |m|(T), C(T), χ(T) のプロット（エラーバー付き）
- 厳密解（Onsager解）との重ね描き

#### F-VIS-03: ビンダーキュムラントの可視化
- 複数の格子サイズ L での U_L(T) の重ね描きと交差点の明示

#### F-VIS-04: 有限サイズスケーリングプロット
- log-log プロットによる臨界指数の推定

#### F-VIS-05: モンテカルロ時系列の可視化
- エネルギー・磁化の時系列プロット（熱平衡化の確認用）

## 3. 非機能要件

### 3.1 性能要件
- L=32, 10000 MCS: 1分以内
- L=64, 10000 MCS: 5分以内
- L=128, 10000 MCS: 20分以内

### 3.2 拡張性要件
- L = 4 から L = 256 まで対応可能
- メトロポリス法以外のアルゴリズム（ウルフ法等）への差し替えが可能な設計
- ポッツモデルなど他の格子模型への拡張を考慮

### 3.3 保守性要件
- FDD原則に基づくモジュール化された構造
- 厳密解（Onsager解）との比較テスト
- 既知の極限（高温・低温）での振る舞いのテスト

### 3.4 再現性要件
- 乱数シードの設定による再現可能性
- 計算結果をJLD2形式で保存

## 4. 技術仕様

### 4.1 開発環境
- **言語**: Julia 1.10 以上
- **必須パッケージ**: Random, Statistics, LinearAlgebra, JLD2.jl, GLMakie.jl, ProgressMeter.jl

### 4.2 データ形式

```julia

Dict(
    "L" => Int,
    "T" => Float64,
    "J" => Float64,
    "h" => Float64,
    "n_mcs" => Int,
    "n_eq" => Int,
    "seed" => Int,

    "magnetization" => Vector{Float64},
)
```

---

## 5. 制約条件

- 温度: T > 0
- 交換相互作用: J ≠ 0
- 格子サイズ: L ≥ 2
- 自然単位系: k_B = 1 として温度を T/J の単位で表示
- ローカルPC実行を想定、シングルスレッド実行を基本
